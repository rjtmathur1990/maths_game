Steps to run the maths game for students from command line:

1. Download the file "mindhouse.py" in your local machine in any particular folder. Go to the terminal and change your directory to that particular folder.
2. Write the command "python3 mindhosue.py" in terminal to execute the code.
3. The game will ask you to enter the name.
4. Next, the questions on multiplications will come, please enter the answer 
5. After answering, you will be asked if you want to play more? If yes, more questions will come otherwise game will stop there.
6. Once the game stops, you will see the number of correct questions answered out of the total asked and the percentage of marks scored.
7. A csv file with your name is generated with the details of all the answers given by you.
